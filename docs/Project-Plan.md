# ThreatDeck Project Plan

## Overview

ThreatDeck is a personal cybersecurity intelligence platform designed to centralize:

- Security News
- Vulnerability Intelligence
- Threat Actor Intelligence
- AI Summaries
- Research and Analysis

## Vision

Transform scattered cybersecurity information into actionable intelligence through a single platform.

## Current Status

Planning Phase

## Initial Goal

Create a dashboard that aggregates cybersecurity news from multiple sources and presents it in a single interface.

## Long-Term Goals

- Vulnerability Tracking
- Threat Actor Profiles
- ATT&CK Mapping
- AI Summaries
- Analyst Lab
- Weekly Intelligence Reports
